# CV Generator

A React-based tool for creating a professional CV — pick a layout, fill in your data, tweak the look, and export to PDF.

## Features

- **Select layout** — choose from available CV templates
- **Add content** — edit your data in `src/data/cv.js` (name, experience, education, courses, etc.)
- **Adjust styles** — customize colors, fonts, and spacing via CSS files in `src/components/`
- **Export as PDF** — print the page and use *Save as PDF* in your browser's print dialog

## Tech stack

React · Vite · Font Awesome · plain CSS

## Getting started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
